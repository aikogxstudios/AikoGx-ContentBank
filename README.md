# AikoGx Content Bank

Banco de ideas de contenido para **AikoGx Studios**.

## Objetivo

Este repositorio existe para que Fagner/Fak y el equipo nunca se queden sin ideas para publicar.

La idea es simple:

**Fagner hace avances en el proyecto -> Aiko organiza la materia prima -> este banco guarda ideas listas o casi listas para redes, web, devlogs, itch.io y comunidad.**

## Para que sirve

- Guardar ideas para TikTok y YouTube Shorts.
- Preparar ideas para Discord.
- Preparar posts para X.
- Preparar temas de devlogs web.
- Preparar updates para itch.io.
- Guardar ideas que necesitan clip, captura o voz.
- Guardar ideas reciclables cuando no haya material nuevo.
- Mantener un objetivo realista de contenido, como 2 videos por semana.

## Regla principal

No todo tiene que ser una gran novedad.

Un buen contenido puede salir de:

- una captura bonita;
- un bug gracioso;
- una comparacion antes/despues;
- una pregunta a la comunidad;
- una idea de carta;
- un trozo de lore sin spoilers;
- una escena del tutorial;
- una prueba visual;
- una decision de diseño;
- una rareza del mundo de Caos Entre Reinos.

## Flujo recomendado

1. Fagner escribe una nota del dia.
2. Aiko convierte esa nota en ideas.
3. Las ideas se guardan en este repo.
4. El compañero de contenido elige una idea.
5. Se graba o prepara el material necesario.
6. Se publica.
7. Se mueve la idea a publicada o reciclable.

## Nota importante

Este repo esta pensado como banco de ideas. Si el repo es publico, no guardar spoilers fuertes, informacion privada, fechas no confirmadas ni estrategias delicadas.
