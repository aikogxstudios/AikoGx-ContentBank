# AikoGx Content Bank

Banco de ideas de contenido para AikoGx Studios.

## Objetivo

Este repositorio existe para que Fak y el equipo tengan ideas constantes para publicar.

La idea es simple: se escriben avances o notas del dia, Aiko los organiza y este banco guarda ideas para redes, web, devlogs, itch.io y comunidad.

## Para que sirve

- Ideas para TikTok y YouTube Shorts.
- Ideas para Discord.
- Posts para X.
- Temas para devlogs web.
- Updates para itch.io.
- Ideas que necesitan clip, captura o voz.
- Ideas reciclables cuando no haya material nuevo.
- Mantener un objetivo realista de 2 videos por semana.

## Flujo recomendado

1. Fak escribe una nota del dia.
2. Aiko convierte esa nota en ideas.
3. Las ideas se guardan en este repo.
4. El equipo elige una idea.
5. Se prepara el clip, captura o texto.
6. Se publica.
7. Se mueve la idea a publicada o reciclable.
